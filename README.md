# Banking-Data-Analysis-Interactive-Dashboard
Banking Data Analysis &amp; Interactive Dashboard project focused on customer segmentation, financial distribution, and risk assessment. Performed EDA using Python and built a Power BI dashboard to analyze deposits, loans, credit scores, and risk categories for data-driven decision-making.

# Project Overview

This project performs Exploratory Data Analysis (EDA) on a banking dataset to analyze customer demographics, financial distribution, and risk segmentation.

An interactive dashboard was built to provide clear financial and customer insights for better decision-making.

# Objective

Analyze customer profile and financial behavior

Study deposits and loan distribution

Evaluate credit score impact on risk

Identify high-risk customer segments

# Tools Used

Python (Pandas, NumPy)

MySQL

Matplotlib & Seaborn

Jupyter Notebook

Power BI (Dashboard Visualization)

# Key Insights

Medium-risk customers form the majority segment.

Higher income customers tend to maintain larger deposits.

Loan exposure is concentrated within specific risk categories.

Credit score significantly influences customer risk level.

# Business Impact

This dashboard enables banks to:

Monitor financial exposure

Improve credit approval strategies

Identify potential defaulters early

Strengthen customer risk profiling
